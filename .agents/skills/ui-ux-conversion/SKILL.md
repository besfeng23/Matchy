# UI + UX Conversion

Rules:
- Do not dump every feature onto one page.
- Every screen has one primary user and one primary action.
- Show account state, action cost, and disclosure context where relevant.
- Keep report, block, cancel, and help paths visible where they matter.

Done:
- Screen hierarchy is clear.
- Mobile layout works.
- Critical actions have confirmation.
- Error states are useful.
